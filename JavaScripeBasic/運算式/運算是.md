## 數學運算是
+ => 加法
- => 減法
* => 乘法
/ => 除法
% =>  取餘
++ => 遞增
-- => 遞減

## 賦值運算子
=   => 賦值
+=  => 加/ 賦值(x += y) ( x = x + y)
-=  => 減/ 賦值(x -= y) ( x = x - y )
*=  => 乘/ 賦值(x *= y)  ( x = x * y )
/=  => 除/ 賦值( x /= y) ( x = x / y )
%=  => 取於/ 賦值 ( x %= y ) == (x = x % y)

# 比較運算子
== => 相等
=== => 全等
!=  =>  不等
>   =>  大於
>=  =>  大於等於
<   =>  小於
<=  =>  小於等於

# 邏輯運算子
&&  =>  與
||  =>  或
!   =>  非

# 位元運算子
&  =>  與
|  =>  或
~  =>  非
^  =>  異或
<< =>  左移
>> =>   右移
